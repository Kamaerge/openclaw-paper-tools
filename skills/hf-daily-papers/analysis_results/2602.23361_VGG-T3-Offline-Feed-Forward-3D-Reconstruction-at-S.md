## 📄 摘要

We present a scalable 3D reconstruction model that addresses a critical limitation in offline feed-forward methods: their computational and memory requirements grow quadratically w.r.t. the number of input images. Our approach is built on the key insight that this bottleneck stems from the varying-length Key-Value (KV) space representation of scene geometry, which we distill into a fixed-size Multi-Layer Perceptron (MLP) via test-time training. ## VGG-T³ 论文分析 (2602.23361)

### 1. 核心贡献

VGG-T³ (Visual Geometry Grounded Test Time Training) 提出了一个**线性复杂度的离线前馈3D重建模型**。核心创新在于：将传统方法中随图像数量**二次增长**的 Key-Value (KV) 空间表示，**压缩**为固定尺寸的 MLP（多层感知机），通过**测试时训练 (TTT)** 在推理时优化这个 MLP。从而实现 O(n) 线性复杂度，支持大规模图像集合的实时重建。

### 2. 技术方法

- **基础架构**：基于预训练VGGT模型，采用图像tokenizer将多视图图像转换为token序列
- **核心机制**：用**测试时训练**替换全局softmax注意力层
  - **Update阶段**：将KV空间压缩到MLP权重（快速权重）
  - **Apply阶段**：用MLP查询tokens获取输出
- **可扩展性**：支持小批量处理和分布式推理

### 3. 实验亮点

- **速度突破**：1000张图像重建仅需**54秒**，比VGGT快 **11.6倍**
- **大规模处理**：2000×2000图像集合48.5秒完成（VGGT需27分钟，**33倍加速**）
- **精度优势**：点图重建误差显著优于其他线性时间方法
- **新能力**：首次展示同一模型同时支持**建图和视觉定位**

### 4. 相关工作

- **多视图前馈方法**：VGGT、Fast3R、π³、FLARE
- **大场景重建**：SparseVGGT、FastVGGT（Token合并/稀疏注意力，但仍是二次复杂度）
- **在线方法**：StreamVGGT、Stream3R、CUT3R、MUST3R
- **同时期工作**：TTT3R（使用类似TTT机制，但属于自回归模型）
- **经典方法**：COLMAP、Bundler、GLOMAP

### 5. 局限性

- **非纯前馈**：仍需测试时优化迭代，非端到端前馈
- **精度权衡**：相比完整二次复杂度方法可能有精度损失
- **定位初步**：视觉定位能力仅为概念验证
- **初始化依赖**：需要VGGT预训练权重初始化
